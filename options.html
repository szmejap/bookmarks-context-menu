Main bookmarks folder name



Substituted regular expression



The extension will recreate the bookmark folder structure under the Main bookmarks folder. Only the first folder with matching name is taken into account.

Anny occurence of the Substituted regular expression in the URLs of bookmarks will be replaced with the selection. For example "http://www.example.org/?search=%s" becomes "http://www.example.org/?search=[selected text]", where "[selected text]" is whatever is currently selected.
The default value "%s" corresponds to the way that bookmark keywords work in Firefox. You can read more about this here.

How to create a "context menu bookmark":

  • Create a bookmarks folder with the name indicated in the Main bookmarks folder setting (by default "Context Bookmarks")
  • Find a webpage with a search function e.g. "https://www.google.com/"
  • Use the search function on the webpage to find any string e.g. "searching"
  • Bookmark the result, and put the bookmark in the folder from the first step
  • Edit the bookmark address:
    • If the searched string was "searching", the URL should look like: "https://www.google.com/search?q=searching"
    • Change the URL to "https://www.google.com/search?q=[substituted regular expression]" by replacing "searching" with Substituted regular expression
    • For the default value of Substituted regular expression (i.e. "%s"), the URL should be changed to "https://www.google.com/search?q=%s"
    • Save the bookmark
    • The bookmark will now appear in the context menu
  • The Substituted regular expression can be put anywhere in the URL, multiple times, or even not at all (in that case the selected text will not be passed to the URL)
  • Try bookmarking search engines, dictionaries, code repositories etc.
  • Any subfolder structure under Main bookmarks folder will be reflected in the context menu
  • The bookmarks still behave, like any other bookmark, in particular a keyword may be added to them

How to use a "context menu bookmark":

  • Select a text on a webpage
  • Right click anywhere
  • Use the context menu to select any option under "Search [selection]", where [selection] is the selected text
